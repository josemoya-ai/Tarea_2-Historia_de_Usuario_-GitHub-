# HU-03: Exploración del menú

**Como** cliente de LePetitDep,
**quiero** visualizar el menú de cafés y productos de panadería disponibles,
**para** seleccionar los productos que deseo comprar.

## Criterios de aceptación

* El sistema debe mostrar una lista de productos disponibles.
* Cada producto debe incluir nombre, descripción y precio.
* Los productos deben estar organizados de forma clara.
* El usuario podrá consultar la información de cada producto antes de agregarlo al carrito.
