# HU-06: Historial de pedidos

**Como** usuario registrado,
**quiero** consultar mis pedidos anteriores,
**para** revisar mis compras realizadas y sus detalles.

## Criterios de aceptación

* El sistema debe mostrar todas las órdenes realizadas por el usuario.
* Cada pedido debe mostrar fecha, productos y monto total.
* El usuario podrá consultar los detalles completos de una orden específica.
* La información mostrada debe corresponder únicamente al usuario autenticado.
