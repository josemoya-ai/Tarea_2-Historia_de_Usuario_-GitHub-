# HU-05: Realizar pedido

**Como** cliente,
**quiero** confirmar mi pedido y seleccionar un método de pago simulado,
**para** generar una orden de compra para recoger en el local.

## Criterios de aceptación

* El sistema debe mostrar un resumen del pedido.
* El usuario debe seleccionar un método de pago.
* El sistema debe generar un número único de orden.
* El usuario debe recibir una confirmación del pedido realizado.
* El pedido debe almacenarse en el historial del usuario.
