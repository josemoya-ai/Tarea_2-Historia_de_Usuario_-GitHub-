# HU-04: Agregar productos al carrito

**Como** cliente,
**quiero** agregar productos y definir cantidades en mi carrito de compras,
**para** preparar un pedido antes de realizar el pago.

## Criterios de aceptación

* El usuario podrá seleccionar uno o varios productos.
* El usuario podrá indicar la cantidad deseada.
* El carrito mostrará el subtotal y el total actualizado.
* El usuario podrá modificar o eliminar productos del carrito.
