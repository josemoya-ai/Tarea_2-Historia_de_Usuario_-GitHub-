# HU-02: Inicio de sesión

**Como** usuario registrado,
**quiero** iniciar sesión con mi correo electrónico y contraseña,
**para** acceder a mi cuenta y gestionar mis pedidos.

## Criterios de aceptación

* El sistema debe validar las credenciales ingresadas.
* Si las credenciales son correctas, se permitirá el acceso.
* Si las credenciales son incorrectas, se mostrará un mensaje de error.
* El sistema debe mantener la sesión activa mientras el usuario navega.
