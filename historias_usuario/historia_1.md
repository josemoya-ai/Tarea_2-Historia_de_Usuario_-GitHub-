# HU-01: Registro de usuario

**Como** visitante de LePetitDep,
**quiero** crear una cuenta utilizando mi correo electrónico y una contraseña,
**para** acceder a las funcionalidades de la plataforma y realizar pedidos.

## Criterios de aceptación

* El sistema debe solicitar correo electrónico y contraseña.
* El correo electrónico debe ser único.
* La contraseña debe cumplir requisitos mínimos de seguridad.
* El sistema debe confirmar el registro exitoso.
* El usuario podrá iniciar sesión después del registro.
