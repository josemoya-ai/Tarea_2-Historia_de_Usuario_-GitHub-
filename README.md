# Tarea_2-Historia_de_Usuario_-GitHub-
Aplicación web ficticia "LePetitDep", permite a los usuarios pedir café y productos de panadería para recoger en su local.

Cada funcionalidad inicia con un flujo guiado:

- Registro de usuario: crear cuenta con email y contraseña.
- Inicio de sesión: acceso seguro con validación.
- Explorar menú: ver lista de cafés, precios y descripciones.
- Agregar productos al carrito:  seleccionar cantidad y productos.
- Realizar pedido: confirmación, método de pago simulado y número de orden.
- Historial de pedido: consultar órdenes anteriores.
